PROJECT


PENNY TRACKER is an all-in-one financial ecosystem designed to transform how you interact with your money. Moving beyond simple logging, it serves as a strategic partner in wealth building. By integrating real-time tracking of expenses, automated budgeting, and investment growth visualization ,PENNY TRACKER gives you a 360-degree view of your net worth.


DESCRIPTION


The Growth Engine: Visualize your savings and investments through dynamic projections. Watch your future wealth grow as you hit your monthly contribution goals.

Precision Budgeting: Set granular limits across categories and receive proactive alerts before you overspend.

The "Burn" Protocol (The Roast): Our signature AI-powered feedback system. If you spend 500 on takeout while your "Savings" goal is lagging, expect a witty, brutal roast to keep your discipline sharp.

Spending Optimization: Identify recurring "leaks"—those forgotten subscriptions or habit-based expenses—and get actionable tips on how to minimize waste.

Frictionless UX: A minimalist, dark-themed interface designed for speed, ensuring that logging a transaction takes less than three seconds.

TECH STAK

1. Core Framework: React (JavaScript/TypeScript)

2. Styling: Tailwind CSS

3. Data Visualization: Recharts or Lucida-React

4. State Management: React Hooks (us estate / use Memo)

5. Data Persistence: Browser Local Storage


SETUP INSATRUCTIONS



1. Environment Setup
Before opening the code, make sure you have Node.js installed.

Download: nodejs.org (LTS Version).

This provides the npm command needed to run the project.


2. Steps to Run the Website
Step 1: Open the Project in VS Code
Launch Visual Studio Code.

Go to File > Open Folder... and select the root folder of the project (the one containing package. Son).

3: Install Dependencies
You must download the necessary libraries (like React and Tailwind) that aren't included in the file transfer to save space. Run this command:

Bash
npm install
This will create a node modules folder. It might take a minute depending on your internet speed.

Step 4: Launch the Development Server
Once the installation is complete, start the local server:

Bash
npm run dev


Step 5: View the Site
Look at the terminal output. It will show a link like http://localhost:5173.

Ctrl + Click that link to open your browser and see the Expense Tracker in action.


3. Troubleshooting for New Users
"Command not found: npm": This means Node.js is not installed or your terminal needs a restart.

Missing .env files: If your project uses API keys, remind them that these files are often hidden and need to be shared separately.

Port already in use: If 5173 is busy, Vite will automatically move to 5174. Always check the terminal for the active link.
